Tämä on tämmöinen Pythonilla koodattu pairingsien generointisofta, ohjelma
käynnistyy esim. komennolla python gui.py. Käyttöliittymä on tehty 
wxPythonilla, joten se pitää myös olla asennettuna. Moduuli mwmatching.py
on Joris van Rantwijkin tekemä, kyseisen henkilön kotisivut ovat osoitteessa
http://jorisvr.nl/, muuten olen koodannut ohjelman itse. Yhteystiedot:
